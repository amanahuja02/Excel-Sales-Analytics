# Sales & Financial Analysis Reports

This folder contains PDF reports generated from Excel-based analysis projects. Each report focuses on a specific business problem commonly handled by data analysts, such as profitability analysis, market performance evaluation, and scenario planning.

---

## Report 1: Profit & Loss (P&L) Analysis

File: p_and_l_fine_tuning.pdf

### Objective
To analyze profitability across countries by evaluating revenue, costs, and margins, enabling stakeholders to identify high-performing and low-performing regions.

### Business Problem Solved
Organizations need a clear view of where profits are generated and where costs are eroding margins. This report helps decision-makers assess regional financial health and optimize operations.

### Key Metrics
- Net Sales  
- Cost of Goods Sold (COGS)  
- Gross Margin  
- Gross Margin Percentage (GM%)

### Skills Demonstrated
- Financial statement analysis  
- Pivot table reporting  
- Profitability analysis  
- Business interpretation of financial data

### Key Metrics

- Discounts  
- Cost of Goods Sold  
- Gross Margin vs Target  

### Skills Demonstrated
- Scenario modeling  
- What-if analysis  
- Forecasting and planning  
- Decision-support analytics
