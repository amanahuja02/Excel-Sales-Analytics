Sales and Financial Analysis Reports (PDF)

This folder contains exported PDF reports generated from the Excel Sales and Profitability Analysis project. These reports present finalized insights derived from Excel dashboards, pivot tables, and financial models.

Reports Included

1. Profit and Loss (P&L) Report  
Provides country-wise financial performance analysis including Net Sales, Cost of Goods Sold (COGS), Gross Margin, and Gross Margin percentage. Useful for evaluating regional profitability.

File: p_and_l_fine_tuning.pdf

2. Sales Market Performance Report  
Presents year-over-year sales comparisons and product-level performance analysis to evaluate growth trends and market performance.

File: sales_market_report.pdf

3. Planning and Scenario Analysis Tool  
Includes best-case, realistic-case, and worst-case scenarios to analyze the impact of discounts, cost structures, and sales volume on profitability.

File: planning tool.pdf

4. Advanced Analytics Practice Report  
Contains additional analytical views and calculated metrics created to practice advanced data analysis concepts.

File: movies_more_dax.pdf

Tools Used

Microsoft Excel  
Pivot Tables and Filters  
Financial Modeling  
PDF Export for Reporting

Purpose

These reports demonstrate practical business analytics skills including financial analysis, sales performance evaluation, profitability assessment, and structured reporting for stakeholders.
